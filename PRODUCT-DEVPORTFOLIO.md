# DevPortfolio Builder - Product Spec

## Problem
- 60K+ tech layoffs in March 2026 (Meta alone: 16,000)
- Hiring managers NOT reading resumes - want projects, skills context, scale
- Laid off devs need quick path to hired
- 34,742 applicants for single MERN/React role in Bangalore

## Solution
Automated portfolio builder that generates:
1. GitHub repository with star-worthy projects
2. Deployed application (hosted demo)
3. Technical blog with design decisions

## Features
- Project templates (API, full-stack, data)
- Deployment automation
- Blog post templates explaining decisions
- README generators

## Target
Laid-off developers needing quick portfolio for job search

## Status
MVP READY - Core generation scripts working
