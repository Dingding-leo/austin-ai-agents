# DevPortfolio Builder

Generate a professional developer portfolio in 3 minutes.

## Quick Start

1. Open `portfolio-builder.html` in your browser
2. Fill in your details
3. Download your portfolio

## Features

- ⚡ Generate in 3 minutes
- 🎯 Interview-ready projects
- 🚀 One-click deployment ready
- 💯 100% Free

## Deployment Options

### GitHub Pages (Free)
```bash
# 1. Create a new repo on GitHub
# 2. Upload your portfolio.html
# 3. Go to Settings → Pages → Deploy from main branch
```

### Netlify (Free)
1. Drag and drop your portfolio folder
2. Get instant HTTPS link

### Vercel (Free)
```bash
npm i -g vercel
vercel
```
