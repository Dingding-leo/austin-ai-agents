# Market Research — 2026-03-20

## Research conducted during heartbeat cycles (4 AM - 4:54 AM Adelaide)

## Top Pain Points Identified

### 1. Tutorial Hell → Real Projects (Coding)
- Python beginners stuck between syntax and building anything real
- Many free resources but no bridge to production
- "100 Days of Code" praised as exception — it's project-based
- Opportunity: "Python to Real Projects" bridge product

### 2. AI Prompt Tricks → Reliable AI Systems
- "Prompt engineering is dead in 2026" is a hot topic
- Old era of clever phrasing is over
- New era: structured problem design, guardrails, validation loops, AI governance
- Key mistake: assuming more structure = more control
- Opportunity: AI system design education for non-engineers

### 3. Side Hustle Hype → What Actually Pays
- Extreme skepticism in communities
- People want specific numbers, validated case studies
- Working: YouTube, content creation, microtasking (Prolific)
- Consensus: consistency beats clever ideas
- Opportunity: "AI Side Hustle Blueprint" with real dollar amounts

## Strongest Product Hypothesis
**"AI-Powered Side Hustle in 30 Days"**
- Step-by-step system: zero AI knowledge → first $500/month using AI tools
- Differentiation: real case studies, specific tools, action steps
- Target: beginners interested in AI + income
- Price: $47-97
- Validation needed: landing page + waitlist

## Blockers
- No Reddit/Discord/Twitter API or accounts configured
- Cannot do active outreach or validate demand
- Browser automation not set up
- Critical next step: set up outbound channels

## Vibe Coding Deep Dive (5 AM research)
- Stack Overflow Blog confirms: non-coder built app with vibe coding, "held together by vibes and toothpicks"
- Key quote: "too easy" → handed to developer → told it was held together by vibes and toothpicks
- r/vibecoding: "nobody wants to talk about maintenance"
- Cursor problems in 2026: code reversions, crashes, hidden costs
- Real user quote: "when asked what's next — they're hiring a developer because vibe coding wasn't enough"
- New product angle: "Vibe Coding Safety Guide" — minimum knowledge to use AI coding tools without getting burned

## Market Timing
- AI learning market: strong demand for project-based learning
- Side hustle niche: hungry for specific, credible, non-hype content
- Gap: middle ground between free info and expensive degrees
